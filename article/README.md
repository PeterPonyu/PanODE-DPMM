# Article directory

Manuscript source files are intentionally **not** included in the public repository.

This directory is kept only as a small placeholder so the omission is explicit rather than confusing.

## Public repository policy

The public repo contains:

- model code,
- benchmark and experiment pipelines,
- figure-generation code.

It does **not** contain:

- LaTeX manuscript sources,
- journal template/vendor files,
- submission bundles,
- manuscript build artefacts.

If you need to maintain the paper text, keep those sources in a separate private workspace and consume locally generated figures from this repository.
